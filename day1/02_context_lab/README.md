# 실습 2 - 나만의 코딩 어시스턴트 만들기

## 목표
시스템 프롬프트와 지식 파일을 직접 작성해서,
**나의 업무에 최적화된 AI 코딩 어시스턴트**를 만들어봅니다.

## 파일 구조
```
02_context_lab/
├── app.py            # 챗봇 앱 (수정 불필요)
├── system_prompt.md  # 챗봇 페르소나 및 답변 방식 정의
├── knowledge.md      # 나의 기술 스택 및 업무 맥락 (직접 작성)
└── README.md         # 실습 가이드
```

## 실습 순서

### Step 1. 설치 및 실행
```bash
pip install openai streamlit
streamlit run app.py
```

### Step 2. 지식 파일 없이 먼저 질문해보기
`knowledge.md` 를 비워둔 채로 아래 질문을 해보세요.
```
Spring Boot에서 JWT 로그인 API를 만들어줘
```
→ 결과를 기억해두세요.

### Step 3. knowledge.md 직접 작성
본인 기술 스택과 컨벤션을 `knowledge.md` 에 채워보세요.
최대한 구체적으로 작성할수록 좋습니다.

### Step 4. 같은 질문 다시 해보기
사이드바의 **🔄 프롬프트 새로고침** 버튼을 누르고 같은 질문을 해보세요.
→ Step 2와 어떻게 달라졌나요?

### Step 5. system_prompt.md 수정해보기
`system_prompt.md` 하단의 자유 수정 구역을 바꿔보세요.
- 어시스턴트 이름 바꾸기
- 답변 규칙 추가하기

### Step 6. 도전 과제
아래 질문들로 어시스턴트를 테스트해보세요.
- "내 프로젝트에서 자주 쓰는 패턴으로 CRUD API 만들어줘"
- "이 코드 리뷰해줘 (코드 붙여넣기)"
- "N+1 문제 해결 방법 알려줘"

---

## 핵심 포인트
knowledge.md 를 더 구체적으로 쓸수록 → 더 좋은 답변이 나옵니다.
이것이 바로 **컨텍스트 엔지니어링** 입니다.
