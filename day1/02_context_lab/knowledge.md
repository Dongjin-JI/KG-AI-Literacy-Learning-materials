# 코딩 튜터봇 지식 베이스

============================================================

이 파일을 채울수록 챗봇이 내가 가르치는 학생에게 맞는 답변을 줍니다.
아래 항목을 상황에 맞게 직접 작성해보세요.

[상황]
당신은 코딩 과외를 하고 있는 선생님입니다. 학생이 자율학습 할 때
도움이 되는 코딩 튜터 챗봇을 만들려고 합니다. 기초 코딩 문법과
기초 코딩테스트(알고리즘) 관련 자료를 학생 수준에 맞게 잘 전달하는 것이
목표입니다.

[작성 방법]
아래 항목은 Continue Chat을 **Chat 모드**로 열어서 함께 채워보세요.
혼자 채우기 막막하면 이렇게 물어봐도 됩니다.
"나는 코딩 과외 선생님이고, 학생 자율학습용 코딩 튜터 챗봇을 만들려고 해.
아래 항목들을 채우는 것 좀 같이 도와줘: (항목 목록 붙여넣기)"

[목적/사용 시점]
이 파일은 오늘(Day1) 챗봇 실습에서만 쓰고 끝나는 파일이 아닙니다.
이렇게 지식 파일을 구체적으로 작성하는 연습은 Day2 "Rules"
실습에서 프로젝트 규칙을 정의할 때도 동일하게 활용됩니다.

[보안 안내]
실제 학생 이름, 학교명 등 개인정보는 그대로 적지 말고,
패턴만 일반화해서 작성하세요.
(예: "OO고 1학년 김OO" (X) → "고등학교 1학년, 코딩 입문 단계" (O))

## 튜터링 대상 (필수)
- 가르치는 대상:      # 예: 코딩 입문 고등학생 (또는 비전공 성인 학습자, 대학 신입생 등)
- 주로 다루는 언어:    # 예: Python (또는 JavaScript, C++ 등)
- 학생 수준:          # 예: 변수/조건문/반복문 막 배운 초급자 (또는 자료구조 학습 중인 중급자 등)

## 커리큘럼 범위 (선택)
- 기초 문법 범위:      # 예: 변수, 조건문, 반복문, 함수, 리스트/딕셔너리
- 코딩테스트 준비 범위: # 예: 완전탐색, 정렬, 이분탐색, 그리디, DP 기초
- 목표 일정:          # 예: 3개월 뒤 코딩테스트 응시 예정 (또는 특정 학교 과제 대비 등)

## 설명 스타일 (필수)
- 설명 순서:          # 예: 개념 설명 → 힌트 → (요청 시) 정답 코드 순으로
- 코드 스타일:        # 예: 들여쓰기 4칸, 변수명은 의미 있게 짓기
- 주석 스타일:        # 예: 코드 한 줄마다 설명 주석 달기

## 자주 쓰는 지도 패턴 (필수)
예시:
- 문제 풀이 전에 관련 개념부터 짧게 정리해주기
- 학생 코드에 오류가 있으면 바로 고치지 말고 원인을 먼저 질문하기
- 유사한 연습 문제를 1~2개 더 제시해서 반복 연습시키기

## AI에게 원하는 답변 방식 (필수)
예시:
- 정답 코드는 학생이 "정답 보여줘"라고 명확히 요청할 때만 제공해줘
- 어려운 용어는 쉬운 비유를 들어 설명해줘
- 코드에는 한 줄씩 설명 주석을 달아줘

## 학생들이 자주 헷갈려하는 부분 (선택)
예시:
- 재귀 함수 개념
- 시간복잡도 계산
- 인덱스 범위 오류 (off-by-one)
- 재귀와 반복문 중 언제 무엇을 써야 하는지
